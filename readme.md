# [MyFeed](http://www.myfeed.cool) - The best place to discover, discuss, and share articles with friends.

## Getting Started

1. `git clone https://github.com/noahmacca/my-feed.git`
2. `npm i`
3. `mongod` (run mongo locally)
4. `export PORT=8888`
5. `node app.js`

## Bugs and Issues
